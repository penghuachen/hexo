迴圈

透過條件的限定，當條件符合時，重複執行某段程式碼。

while

條件符合時會執行迴圈中的內容

do/while

程式碼會先執行一次後在依照條件判斷是否要繼續執行

for

跳躍句

break

作用是跳離switch或迴圈(當迴圈沒必要完全執行完成時)

continue

類似break，但break是跳離迴圈，而continue會中斷不符合條件的部分，並往下繼續執行

return

  • 只能出現在函式中
  • 使用時會回傳運算式的值給caller
  • 在函式中任意位置都可以使用return，但要注意的是，當程式碼執行到return處就會回到呼叫者處。
  • 可以用於中斷函式的進行。
