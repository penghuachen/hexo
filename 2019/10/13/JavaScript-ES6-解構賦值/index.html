JavaScript ES6 解構賦值

MDN : 可以將陣列或物件中的資料取出成獨立變數。

解構賦值是ES6的新特性，可以快速的將值從陣列或物件中取出。

如果能好好熟悉運用的話，不僅快速、更可以提高程式碼的可讀性。

趕緊往下看吧!

物件解構
