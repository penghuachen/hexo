完賽日，心得與階段學習驗收

今年度的鐵人賽於今天即將完賽，終於要告一段落啦!!!

回顧這三十天的的學習，給自己打了個差強人意的分數。

不過整體來說還是把 React 的基礎算是有了個基本的了解

而本次給自己訂定的目標，算是有成功達標，真的是小確幸XD

不過該如何運用到實務經驗中才是考驗的開始，填坑之路才正要前行!

而今天的篇幅最後附上作為驗收的小小 demo 以及本次的學習文章目錄。

簡單說明一下這個 demo 是一個非常簡單的照片牆，透過串接 Unsplash API 的資料來完成。

而為了可以將這段時間學到的觀念運用到，所以使用了 React Redux 來管理狀態，至於在頁面切換的部分則是透過 React Router 完成。

雖然只是一個非常簡單的 demo，但還是跑過了 React 基本該使用的那些觀念與套件。

最後，期待明年可以再相見(?!)
